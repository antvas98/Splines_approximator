# Splines_approximator

Approximate the function  $f(x)= \frac{\sin(12(x+0.2))}{(x+0.2)}$ using natural cubic splines as described in in Chapter 5.2.1 of [The Elements of Statistical Learning](https://link.springer.com/book/10.1007/978-0-387-84858-7).
